# Product Overview

地元紹介 × 名刺アプリ — 目の前の人との会話を生む装置

## Core Capabilities

1. **名刺生成**: 出身地を入力 → AIが地域特有の議論ネタを生成 → ユーザーが立場を選択 → 地元名刺が完成
2. **名刺交換**: スマホ同士をぶつけて交換（DeviceMotion API + WebSocket）。Webブラウザだけで完結
3. **比較・会話創出**: 2人の名刺を並べて一致/不一致を可視化 → その場で会話が生まれる

## Target Use Cases

- 地方出身者が新生活で新しい土地に行き、友達に「地元どこなん？」と聞かれた際に使う
- 初対面の対面シーンで、SNSに依存せずに会話のきっかけを作る
- 新生活シーズン（大学入学、就職）での自己紹介・交流

## Value Proposition

**SNSを頑張っていない人でも、初対面の会話を楽しめる。**

- 出身地さえあれば会話のきっかけが自動で生まれる（投稿ゼロでもOK）
- 連絡先ではなく「会話のきっかけ」を交換する
- すれ違い通信のワクワク感をWebブラウザだけで再現
